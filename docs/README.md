## <img src="https://img.icons8.com/fluency/48/open-book.png" width="28" align="absmiddle"/> Introduction

**PixelPatch** is your ultimate tool for photo privacy protection and intelligent image markup. Whether you're blurring sensitive information, masking text, or adding customized mosaic effects, PixelPatch makes it fast, elegant, and secure.

### <img src="https://img.icons8.com/fluency/48/star.png" width="24" align="absmiddle"/> Key Features

- **Smart Text Recognition (OCR)**: Automatically detect and mask sensitive text strings in seconds.
- **Mosaic & Pixelation Tools**: Seamlessly blur faces, license plates, and private documents.
- **Dynamic Adaptability**: Fully supports dark mode and multiple screen layouts.
- **Lossless Export**: Save edited photos natively without quality degradation.
