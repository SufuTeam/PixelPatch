## <img src="https://img.icons8.com/fluency/48/open-book.png" width="28" align="absmiddle"/> 製品について

**PixelPatch** は、写真のプライバシー保護とインテリジェントな画像マークアップのための究極のツールです。機密情報のぼかしやカスタムモザイク効果の追加など、PixelPatchを使用すれば、素早く、エレガントに、そして安全に処理を行うことができます。

### <img src="https://img.icons8.com/fluency/48/star.png" width="24" align="absmiddle"/> 主な機能

- **スマートテキスト認識 (OCR)**：写真内の機密テキストを数秒で自動検出し、マスク処理します。
- **モザイク＆ピクセル化ツール**：顔、ナンバープレート、プライベートな文書をシームレスにぼかします。
- **ダイナミック適応**：ダークモードとマルチスクリーンレイアウトを完全にサポート。
- **ロスレス書き出し**：画質を損なうことなく、編集した写真を元の品質で保存できます。

**v1.2.0 (2026-03-26)**
- 完全な多言語サポート（英語、日本語、繁体字中国語）を追加しました。
- 画像読み込み時の UI のカクつきを解決するため、レンダリングパフォーマンスを最適化しました。
- ダークモードの体験を向上させ、縦長画像の判定と操作ロジックを最適化しました。

**v1.1.0 (2026-03-22)**
- Jetpack Compose を使用して、画像マークアップ画面 (`DiscernScreen`) を完全にリファクタリング。
- 長押しによるより正確なテキスト領域の選択＆モザイク機能を追加しました。
- EXIF 情報の表示機能、設定画面、そして新しい統合された下部アクションバーを追加しました。

**v1.0.0**
- 初回リリース。OpenCV ベースのテキスト検出システムと、手動モザイクブラシツールを提供。
