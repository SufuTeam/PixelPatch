**PixelPatch (像素打码)** 是保护您图片隐私和进行智能图像标注的终极工具。无论是对敏感信息进行模糊处理，还是添加自定义马赛克效果，PixelPatch 都能让一切变得快速、优雅且安全。

### ✨ 核心功能
- **智能文字识别 (OCR)**：秒级自动检测并遮挡敏感文字内容。
- **马赛克与像素化工具**：无缝模糊人脸、车牌及私人文档。
- **动态主题适配**：全面支持夜间模式（Dark Mode）与多屏幕布局。
- **无损导出**：以原生画质保存编辑后的照片。
